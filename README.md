# Charity: water Game Prototype

In this project, you’ll begin transforming your game concept from last week into a working interactive prototype using HTML, CSS, and JavaScript.

This first version should focus on core functionality — clickable elements, score tracking, and a basic layout. The goal is to bring your idea to life with simple, working mechanics that you’ll refine and expand in the next milestone.

## 

### Rubric
|Criteria|Requirements|Pts|
| --- | --- | --- |
| Core Interactivity | Student implemented at least one working interactive element that updates the game state, AND the player's score is tracked and displayed. Examples: clicking a button increases score; tapping an image shows a message; collecting a drop changes the UI | 25pts |
| Responsive Layout | Layout adjusts well for both desktop and mobile screen sizes. Elements remain usable and visually clear | 15pts |
| Use of Branding & Visual Design | charity: water brand elements (logo, colors, icons) are used thoughtfully to support the game’s theme | 10pts |
| Reflection Questions | Made an effort to answer all three reflection questions | 30pts |
| __(BONUS)__ Add a Challenge | Gameplay includes a working twist (e.g., obstacles, penalties) that enhances the experience | 10pts |
| __(BONUS)__ Game Controls | A working Reset button is included and clearly controls gameplay | 5pts |
| __(BONUS)__ Celebrate Wins | A win condition triggers a clear celebration (e.g., confetti, animation) | 5pts |
